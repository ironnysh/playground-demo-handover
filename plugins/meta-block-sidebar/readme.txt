== Description ==

A custom block that adds two custom meta fields (`team` and `joined`) specifying the person's team name and the date (Month, Year) they joined the company, and display them in the frontend*.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Add the "Meta Block Sidebar" block to the relevant template* (i.e., `/wp-content/themes/blue-note/meta-block-sidebar-post.html`).
4. Fill in the two meta fields in the new section of the sidebar: Team name and the month and year the person joined.

*Note: The template that displays the block "Meta Block Sidebar" on the frondend.
